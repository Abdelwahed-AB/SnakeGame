## Snake Game

#### Description
We all know the classic snake game ... this is my shot at remaking it.

#### Tasks

- [x] Snake movement
- [x] Apple collection
- [x] Snake growth
- [x] Wall collision

#### How to run

```
mvn clean compile
```

change directory to compiled classes

```
cd target/classes
```

Execute main class

```
java snakeGame.Main
```
